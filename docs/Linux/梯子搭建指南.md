#  科学上网之梯子搭建指南

## v2ray

### 服务端安装

1. 一键安装脚本

   ```bash
   sudo bash <(curl -s -L https://git.io/v2ray.sh)
   ```

2. 官方安装

   ```bash
   ## ubuntu
   apt install v2ray
   
   ## centos
   # yum install v2ray
   
   ## manjaro
   # pacman install v2ray
   ```

   配置文件 `/etc/v2ray/v2ray.conf`
